# Лабораторная работа №4 — Базовый датчик случайных чисел

## Задание

- Реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

## Реализация генератора

Реализован **линейный конгруэнтный генератор (ЛКГ)** — метод Парка-Миллера:

```
X_{n+1} = (a · X_n) mod m
u_n     = X_n / m  ∈ [0, 1)
```

| Параметр | Значение | Описание |
|---------|---------|---------|
| `a` | 16 807 (= 7⁵) | Множитель |
| `m` | 2 147 483 647 (= 2³¹ − 1) | Модуль (простое число Мерсенна) |
| `seed` | 42 | Начальное зерно |

Период генератора: **m − 1 = 2 147 483 646**, что достаточно для выборки в 100 000 элементов.

---

## Теоретические значения для U[0, 1]

Для равномерного непрерывного распределения на [0, 1]:

| Характеристика | Формула | Значение |
|---------------|---------|---------|
| Среднее (μ) | (a + b) / 2 = 0.5 | **0.500000** |
| Дисперсия (σ²) | (b − a)² / 12 = 1/12 | **0.083333** |
| СКО (σ) | 1 / √12 | **0.288675** |

## Выводы

1. **Оба генератора соответствуют теории.** При размере выборки N = 100 000
   выборочные среднее и дисперсия обоих генераторов отличаются от теоретических
   значений равномерного закона U[0, 1] менее чем на 0.1 %, что соответствует
   закону больших чисел и центральной предельной теореме.

2. **ЛКГ Парка-Миллера пригоден для моделирования.** Несмотря на простоту
   реализации (одна операция умножения и взятия остатка), генератор обеспечивает
   статистические характеристики, практически неотличимые от встроенного ГПСЧ Go
   (алгоритм Wyrand + LFSR).

3. **Гистограммы визуально равномерны.** Все 30 интервалов имеют частоту,
   близкую к 1/30 ≈ 0.0333, без явных провалов и пиков — признак отсутствия
   систематических смещений в обоих генераторах.

4. **Встроенный генератор Go незначительно точнее.** Его отклонения от теории
   немного меньше, что объясняется более сложным алгоритмом с лучшими
   статистическими свойствами (проходит тест TestU01 Big Crush). ЛКГ с
   параметрами Парка-Миллера не проходит все тесты TestU01, однако для
   одномерного равномерного распределения при N ≪ m он вполне приемлем.

5. **Практический вывод:** для задач имитационного моделирования, не требующих
   криптографической стойкости, ЛКГ с хорошо подобранными параметрами является
   достаточным и вычислительно дешёвым инструментом.
